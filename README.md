# 🎮 Tic-Tac-Toe Game

A simple and interactive Tic-Tac-Toe game built using HTML, CSS, and JavaScript. The game supports both two-player mode and an AI mode where players can challenge an unbeatable computer opponent.

This project was created to practice JavaScript logic, DOM manipulation, and game development concepts while designing a clean and responsive user interface.

---

## ✨ Features

* Play with a friend on the same device
* Play against an AI opponent
* Tracks wins, losses, and draws
* Highlights winning cells
* Responsive and modern UI
* Built using pure HTML, CSS, and JavaScript
* No external libraries or frameworks used

---

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 How to Run

1. Download or clone the repository
2. Open `tictactoe.html` in your browser
3. Start playing

No installation or setup is required.

---

## 🎮 How to Play

* Choose either 2 Player mode or AI mode
* Player X starts first
* Take turns selecting empty cells
* The first player to get 3 marks in a row wins
* If all cells are filled without a winner, the game ends in a draw
